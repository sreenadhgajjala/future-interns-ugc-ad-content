## Project Overview

This project demonstrates AI-powered UGC (User Generated Content) ad creation for a real skincare product. Using prompt engineering and AI tools, I generated multiple hooks, ad scripts, CTAs, and platform-specific content following a Problem → Solution → Result → CTA framework.

The content is designed for Instagram Reels, YouTube Shorts, and Facebook Ads.

# Prompt System

## Master UGC Prompt

Create a 30-second cinematic UGC skincare advertisement featuring a beautiful young Indian woman with natural glowing skin. The advertisement should feel authentic, emotional, relatable, and conversion-focused. Follow a Problem → Solution → Result → CTA structure suitable for Instagram Reels, YouTube Shorts, and Facebook Ads.
